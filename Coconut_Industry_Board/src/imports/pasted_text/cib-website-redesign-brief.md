Prototype Goal:
Shift CIB’s digital presence from mainly informational to more conversion-focused. The website should help users quickly take action, including buying coconut products, making an order inquiry, registering as a grower, calling CIB, getting directions, learning about coconut farming, and engaging with campaign content.

Main Design Requirements:
1. Prioritize mobile-first design, but include desktop/tablet versions where possible.
2. Keep the existing core navigation: Home, About, News & Resources, Registration, Contact.
3. Add a new page or navigation item called “The Coconut Shop” or “Shop Products.”
4. Make key actions visible on mobile instead of hiding all important options inside a side menu.
5. Add quick-access buttons on the homepage and mobile navigation:
   - Shop Products
   - Order on WhatsApp
   - Register as a Grower
   - Call Now
   - Get Directions
   - Learn About CIB
6. Add a sticky bottom mobile navigation bar with icons for:
   - Home

   - Register
   - Directions
